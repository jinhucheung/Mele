Mele
==========================
Mele是仿照外卖o2o平台饿了么设计开发的一个Web App

项目框架
-------------------------
Mele基于`Html/CSS+ajax+servlet+JDBC`实现,数据库使用Mysql

功能说明
-------------------------
- 利用高德地图API实现用户定位
- 过滤距离当前用户位置10km外的商家
- MD5加密用户密码
- 用户管理:用户登录/注册/注销/收货地址
- 商家信息与菜单信息静态写入数据库中
- 实现购物车以及图片轮播功能
